The following button is used to generate the student option entries for any course that the student's are currently enrolled on. The entries that are created are based on the data that is set in the Course List. Before you proceed ensure that all the correct links have been made within the Course List.

Please note:

  • This operation cannot be undone.
  • This operation cannot be repeated.
It is recommended that when the report has loaded that it is saved to an area for the exam officers future used.
